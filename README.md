# Anoncoin website

## Get started

1. Install Ruby
2. Install Bundler: `gem install bundler`
3. Clone this repo
4. Go to project folder: `bundle install`
5. `jekyll serve`
6. Go to: http://127.0.0.1:4000/

Some guide here on how to contribute so that everyone writes nice articles for us.
