The following commands are to be typed in the Help -&gt; debug -&gt; console in anoncoin-qtc or else at the command line to anoncoin-cli, after having started anoncoind

{% highlight bash %}

./anoncoind
Anoncoin server starting
./anoncoin-cli help

{% endhighlight %}
